Selenium WebDriver

Automasi UI menggunakan Selenium (contoh Python)

Apa itu Selenium?

Selenium WebDriver adalah library untuk mengontrol browser secara otomatis — cocok untuk end-to-end tests.

Langkah dasar (Python)

  1. Instal: pip install selenium
  2. Download driver (chromedriver / geckodriver) sesuai versi browser
  3. Tulis script: buka halaman, temukan elemen, interaksi, asersi

Contoh Skrip


from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC

driver = webdriver.Chrome()
driver.get("http://localhost:8000")

wait = WebDriverWait(driver, 10)
weight = wait.until(EC.presence_of_element_located((By.ID, "weight")))
height = driver.find_element(By.ID, "height")
calc = driver.find_element(By.ID, "calculate")

weight.send_keys("55")
height.send_keys("165")
calc.click()

result = wait.until(EC.visibility_of_element_located((By.ID, "result")))
assert "20.2" in result.text

driver.quit()

Tips Sinkronisasi

  • Gunakan explicit waits (WebDriverWait + expected_conditions) bukan sleep()
  • Gunakan page object pattern untuk maintainability
  • Jalankan tests headless di CI

Contoh Page Object (singkat)


class BmiPage:
    def __init__(self, driver):
        self.driver = driver
    def open(self, url):
        self.driver.get(url)
    def set_weight(self, w): self.driver.find_element(By.ID,"weight").send_keys(str(w))
    def set_height(self, h): self.driver.find_element(By.ID,"height").send_keys(str(h))
    def calculate(self): self.driver.find_element(By.ID,"calculate").click()
    def result_text(self): return self.driver.find_element(By.ID,"result").text
