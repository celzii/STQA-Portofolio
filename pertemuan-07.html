Pertemuan-07 — Pengantar Selenium WebDriver

Otomatisasi pengujian antarmuka web menggunakan Selenium

1. Apa itu Selenium dan Selenium WebDriver?

Selenium adalah sebuah framework open-source yang digunakan untuk otomatisasi pengujian aplikasi berbasis web. Selenium memungkinkan pengujian antarmuka pengguna (UI) secara otomatis di berbagai browser seperti Chrome, Firefox, dan Edge.

Selenium WebDriver merupakan komponen utama dari Selenium yang berfungsi untuk mengendalikan browser secara langsung melalui API. Dengan WebDriver, kita dapat membuka halaman web, mengisi formulir, menekan tombol, dan memverifikasi hasil — semuanya dilakukan secara otomatis.

2. Kenapa Harus Selenium?

  • Gratis dan open-source: Mendukung berbagai bahasa seperti Python, Java, C#, dan JavaScript.
  • Multi-browser: Bisa digunakan di Chrome, Firefox, Safari, dan Edge.
  • Cross-platform: Dapat dijalankan di Windows, macOS, maupun Linux.
  • Mendukung otomatisasi kompleks: Termasuk pengujian login, navigasi halaman, form, dan interaksi dinamis.
  • Integrasi mudah: Dapat digabungkan dengan framework testing seperti Pytest atau JUnit untuk CI/CD pipeline.

3. Instalasi Selenium dengan Bahasa Pemrograman Python

Untuk memulai pengujian otomatis dengan Selenium di Python, ikuti langkah berikut:

  1. Pastikan Python telah terinstal di sistem (python --version).
  2. Instal library Selenium dengan perintah: 
    pip install selenium
  3. Unduh WebDriver yang sesuai dengan browser (contoh: ChromeDriver dari chromedriver.chromium.org).
  4. Tempatkan file driver di lokasi yang mudah diakses, misalnya di folder proyek.

4. Membuka Browser dengan Selenium

Berikut contoh sederhana untuk membuka browser menggunakan Selenium: 


from selenium import webdriver
from selenium.webdriver.common.by import By

# Membuka browser Chrome
driver = webdriver.Chrome()

# Membuka halaman web
driver.get("https://www.google.com")

# Menampilkan judul halaman
print(driver.title)

# Menutup browser
driver.quit()

5. Cara Berinteraksi dengan Elemen Halaman HTML

Selenium WebDriver dapat menemukan dan berinteraksi dengan elemen HTML menggunakan berbagai metode seperti id, name, class, tag, atau xpath.

Metode Contoh
find_element(By.ID, "username") Mencari elemen berdasarkan atribut ID.
find_element(By.NAME, "password") Mencari elemen berdasarkan atribut name.
find_element(By.CLASS_NAME, "login-button") Mencari elemen berdasarkan class CSS.
find_element(By.XPATH, "//button[text()='Login']") Mencari elemen berdasarkan path XML/HTML.

6. Contoh Interaksi (Live Coding) Menggunakan Selenium WebDriver

Contoh berikut menunjukkan bagaimana Selenium digunakan untuk membuka halaman, mengisi form login, dan menekan tombol “Login”. 


from selenium import webdriver
from selenium.webdriver.common.by import By
import time

# Membuka browser dan menuju ke situs
driver = webdriver.Chrome()
driver.get("https://www.saucedemo.com/")

# Menunggu beberapa detik agar halaman termuat
time.sleep(2)

# Mengisi username dan password
driver.find_element(By.ID, "user-name").send_keys("standard_user")
driver.find_element(By.ID, "password").send_keys("secret_sauce")

# Menekan tombol Login
driver.find_element(By.ID, "login-button").click()

# Verifikasi bahwa halaman telah berpindah ke dashboard
assert "inventory" in driver.current_url

# Tutup browser
driver.quit()

7. Test Scenario dan Test Case — Situs https://www.saucedemo.com/

Test Scenario

  • Scenario: Pengguna berhasil login ke aplikasi SauceDemo dengan kredensial valid.
  • Precondition: Browser terbuka dan halaman login terbuka.
  • Expected Result: Pengguna diarahkan ke halaman produk setelah login berhasil.

8. Kesimpulan

Selenium WebDriver merupakan alat kuat untuk mengotomatisasi pengujian web. Dengan menggabungkannya bersama framework testing seperti Pytest, kita dapat membuat pengujian berulang, cepat, dan andal. Contoh pada situs SauceDemo menunjukkan bagaimana login otomatis dapat diuji menggunakan skrip sederhana.
